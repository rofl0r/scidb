TITLE Development roadmap

<h2>Features for next releases</h2>

<ul>
  <li>Download/upload/update/load databases via the internet</li>
  <li>Supporting attachments of documents to a database. The main reason
      is that Chessbase databases have attached documents.</li>
  <li>Search functions: search by header, position search, etc.</li>
  <li>Filter functions: game filter, player filter</li>
  <li>Flexible window layout</li>
  <li>Document generation:: LaTeX export</li>
  <li>Windows support</li>
  <li>Input of moves with keyboard</li>
  <li>Board photo</li>
</ul>

<h2>Features for the far future</h2>

<ul>
  <li>Duplicate detection</li>
  <li>Player dictionary dialog</li>
  <li>Document generation: PDF export and HTML export</li>
  <li>Dialog for spell checking player names</li>
  <li>Tip of the Day window</li>
  <li>Generating opening reports</li>
  <li>Usage of end-game tables</li>
  <li>3D chessboard</li>
</ul>

<h2>Volunteers wanted for...</h2>

<ul>
  <li>More languages</li>
  <li>More chess board themes</li>
  <li>More help pages</li>
</ul>

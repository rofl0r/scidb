Der Datenbankumschalter

Der Datanbankumschalter ermöglicht den einfachen Wechsel zwischen Datenbanken. Die aktive Datenbank wird durch einen gelben Hintergrund hervorgehoben.

Mit Halten der mittleren Maustaste (i.d.R. das Mausrad) wird eine Informationsbox angezeigt.

Administration

Zur Administration einer Datenbank steht ein kontextsensitives Menü bereit:

Eigenschaften Öffnet eine Box mit der Anzeige verschiedener Datenbankinformationen.

Export Öffnet einen Dialog zum Datenbankenexport in andere Datenbankformate oder in ein Dokumentenformat.

PGN-Dateien importieren Öffnet den Dateiauswahldialog, um die zu importierenden PGN-Dateien auszuwählen. Es können mehrere PGN-Dateien gleichzeitig gewählt werden. Die Partien werden an die bestehende Datenbank angehängt.

Archiv erzeugen Ein Datenbankarchiv für die gewählte Datenbank anlegen. Der Dateiauswahldialog wird geöffnet, um den Archivnamen zu wählen.

Schliessen Die gewählte Datenbank wird geschlossen. Sind Partien aus dieser Datenbank bearbeitet, aber noch ungespeichert, erfolgt vor der Schliessung eine Sicherheitsabfrage.
Eine Schliessung der Ablage-Datenbank ist nicht möglich.

Komprimieren Diese Datenbank komprimieren, d.h. die zum Löschen vorgemerkten Partien werden physikalisch gelöscht. Diese Aktion kann nicht rückgägngig gemacht werden. Dieser Menüpunkt ist gesperrt wenn keine Partien in dieser Datenbank mit einer Lösch-Markierung versehen sind.

Piktogramm wählen Ein neues Piktogramm für diese Datanbank auswählen.
Dies ist nicht für die Ablage-Datenbank möglich.

Beschreibung bearbeiten Die Beschreibung dieser Datanbank ändern bzw. neu anlegen.
Dies ist nicht für die Ablage-Datenbank möglich.

Schreibgeschützt Diese Datanbank wird schreibgeschützt bzw. schreibberechtigt gesetzt. Dies ist nur für Scidb- und Scid-Datenbanken (.sci, .si4, .si3) möglich.
PGN-Dateien und ChessBase-Datenbanken sind immer schreibgeschützt.
Die Ablage-Datenbank kann nicht schreibgeschützt werden.

Neue Datenbank anlegen Eine neue Datenbank anlegen. Es können nur Scidb- und Scid-Datenbanken (.sci, .si4, .si3) neu angelegt werden.

Datenbank öffnen Eine weitere Datenbank öffnen.

Öffne zuvorige Eine Datenbank öffnen über die Datenbankhistorie.

Symbolgröße Auswahl der Symbolgröße für die Datenbankkacheln.

Hilfe zum Datenbankumschalter Hilfedialog zum Datenbankumschalter öffnen.

Ablage-Datenbank leeren
Dieser Menüpunkt steht nur für die Ablage-Datenbank zur Verfügung.

Drag & Drop

Datenbanken können auch durch Drop (Fallenlassen) einer URI-Liste geöffnet werden. Wird eine URI-Liste (Dateienliste) – oder eine einzelne Datei – in das Datenbankumschaltfenster gezogen, so signalisiert ein farbiger Hintergrund, daß ein Drop erfolgen kann. Auf diese Weise kann das Öffnen von Datenbanken beispielsweise auch mit einem externen Dateimanager erfolgen.

Feldmarkierungspalette

Diese Palette kann auf dem Schachbrettfenster entweder mit dem Tastaturkürzel Strg+M oder über das kontextsensitive Menü des Partieneditors (⇒ Feldmarkierungspalette) geöffnet werden, jedoch nur wenn auch eine Partie im Partieeditor berabeitet wird.

Die Palette erlaubt die Auswahl der Farbe der Markierung, sowie den Markierungstyp. Die Farb- und Typauswahl wurde identisch zur Schachdatenbankapplikation Scid  gehalten.

Die Palette ist nur sichtbar, wenn das Schachbrettfenster aktiv ist.

Im Partieeditor werden Markierungen mit dem Symbol  gekennzeichnet. Die Kurzinfo (Tooltip) zeigt den Code an, wie er im PGN-Format aussieht.

Im folgenden wird beschrieben wie Markierungen gesetzt werden. Das Setzen einer Markierung auf dem Schachbrett ist nur möglich, wenn eine Partie im Partieeditor bearbeitet wird.

  • Ein Feld mit einem Symbol markieren
  • Einen Zug mit einem Pfeil markieren

Ein Feld mit einem Symbol markieren

Ein Doppelklick bzw. ein zweifacher Klick auf ein Feld bei gehaltener Strg-Taste markiert dieses Feld mit dem ausgewähltem Symbol in der ausgewählten Farbe. Die Rücknahme der Markierung geschieht durch erneuten Doppelklick bei gehaltener Strg-Taste mit dem gleichen Symbol und der gleichen Farbe. Es ist zu beachten das nach dem ersten Klick die Markierung für das ausgewählte Feld zu sehen ist, und erst nach dem zweiten Klick wird das Symbol sichtbar. Eine Feldmarkierung kann durch eine andere Feldmarkierung überschrieben werden. Eine mehrfache Feldmarkierung ist nicht möglich.

Einen Zug mit einem Pfeil markieren

Ein Pfeil wird gesetzt durch einen Klick bei gehaltener Strg-Taste auf dem Ausgangsfeld (es ist nun die Markierung für das ausgewählte Feld zu sehen) und einem zweiten Klick bei gehaltener Strg-Taste auf dem Zielfeld. Der Pfeil wird in der ausgewählten Farbe angezeigt. Die Rücknahme einer Zugmarkierung geschieht durch erneutes Setzen eines Pfeils der gleichen Farbe auf den gleichen Ausgangs- und Zielfeldern. Eine Zugmarkierung kann durch eine Zugmarkierung einer anderen Farbe ersetzt werden.

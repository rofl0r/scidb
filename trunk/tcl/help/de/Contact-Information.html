Homepage

Die Scidb-Internetseite ist erreichbar unter http://scidb.sourceforge.net/.

Mithilfe

Gesucht werden Interessenten für:
  • Testen von Scidb
  • Übersetzungen in andere Sprachen
  • Erstellen von Hilfeseiten
    •  Interessenten melden sich bitte mit Hilfe einer Nachricht, damit dies koordiniert werden kann.

    So kann eine Nachricht gesendet werden:

    1. Eine Mitgliedschaft bei Sourceforge ist erforderlich (Account Registration). Die Mitgliedschaft ist Gebuehren- und Verpflichtungsfrei.

    2. Send an Email öffnen, mit dem Mitgliedsnamen einloggen und die Nachricht senden.

    Links

    Für Scidb-Benutzer könnten folgende Seiten interessant sein:

  • Tracker: Feature Requests
    Hier können Funktionswünsche eingetragen werden. Bitte dabei die bestehende Roadmap mit berücksichtigen.

  • Tracker: Bugs
    Hier können Probleme mit Scidb gemeldet werden.

  • Reviews for Scidb
    Diese Seite bietet die Möglichkeit für die Abgabe einer Rezension oder Bewertung über Scidb.

  • Scidb Guestbook
    Hier wird das Gästebuch von Scidb bereitgestellt.

  • Scidb Users Mailing List subskribieren
    Hier kann ein Eintrag in die Mitgliedsliste der "Scidb Users Mailing List" vorgenommen werden. In dieser Liste werden Informationen, News, und Anregungen veröffentlicht. Mitglieder können Meldungen an diese Liste versenden, die dann automatisch an alle Mitglieder verteilt werden.

  • ftp://ftp.cis.uab.edu/pub/hyatt/
    Das legendäre Schachprogram Crafty. Das TB - Unterverzeichnis beinhaltet fast alle Nalimov-Endspieltabellen, welche von einigen Schachprogrammen und auch von Scidb (in einer zukünftigen Version) benutzt werden.

  • tablebase.sesse.net
    Diese Seite bietet ebenfalls Endspieltabellen an, unter anderem auch Sechssteiner.

  • The Week in Chess
    Ein hervorragendes Wochenmagazin für internationale Schachpartien, das PGN Dateien für den Download bereitstellt.

  • Standard: Portable Game Notation Specification and Implementation Guide
    Der PGN-Standard, geschaffen von Steven J. Edwards im Jahre 1994. Die Webseite erklärt das PGN- und EPD-Format im Detail.

  • ECO-Code
    Eine Übersicht über den ECO-Schlüssel.

Die Ablage-Datenbank

Scidb stellt bereits bei Programmstart eine geöffnete Ablage-Datenbank zur Verfügung, welche sich wie eine normale Datenbank verhält, mit der Ausnahme, daß sie im Hauptspeicherbereich existiert und nicht auf der Festplatte gespeichert wird. Damit hat sie nur ein relative begrenztes Aufnahmevermögen. Der vordefinierte Name für diese Datenbank ist Ablage.

Die Ablage kann nicht geschlossen werden. Es exisitiert eine Menüfunktion zum Leeren der Ablage (⇒ Ablage-Datenbank leeren).

Die Ablage hält die Daten intern im Scidb-Format und ist insofern eine Ausnahme, weil sie Partien verschiedener Schachformen gleichzeitig halten kann. Jedoch können administrative Operationen (⇒ Verwaltung) immer nur auf jeweils eine Schachform angewendet werden.

Textformatierung

Kommentare können formatiert werden: Fettschrift, Kursivschrift, und Unterstrichen. Es sind alle Kombinationen dieser Formatierungen erlaubt. Das Setzen einer Formatierung wirkt auf den selektierten Bereich.

Diese Formatierungen können nur im Scidb-Datenbankformat gespeichert werden. Bei der Konvertierung in das Scid-Format, oder beim Export in das PGN-Format (Ausnahme: eine der Optionen Schreibe Kommentar im HTML-Format oder Scidb-Import-Format benutzen wird verwendet) gehen alle Formatierungen verloren.

Tastaturkürzel

Im Textfeld dienen die nachfolgenden Tastaturkürzel zum Einfügen einer Figur:

StrgUmschaltK ♔
StrgUmschaltD ♕
StrgUmschaltT ♖
StrgUmschaltL♗
StrgUmschaltS♘
StrgUmschaltB ♙
Ausserdem sind folgende Standardtastaturkürzel an das Textfeld gebunden:
StrgC Markierten Text in die Zwischenablage kopieren
StrgX Markierten Text löschen und in die Zwischenablage kopieren
StrgV Text aus der Zwischenablage an der Cursorposition einfügen
StrgZ Letzte Aktion im Textbereich zurücknehmen
StrgY Vorhergehende Rücknahme wieder aufheben

Zeilenumbruch

Eine besondere Bedeuting hat ⏎, mit dieser Taste können Zeilenumbrüche erzwungen werden. Im Textfeld werden die erzwungenen Zeilenumbrüche mit dem Absatzsymbol ¶ gekennzeichnet.

Kontextsensitives Menü

Das kontextsensitive Menü im Textfeld bietet folgende Funktionen:

Kopieren Markierten Text in die Zwischenablage kopieren. Dieser Menüpunkt steht nur zur Verfügung, wenn Text markiert ist.

Ausschneiden Markierten Text löschen und in die Zwischenablage kopieren. Dieser Menüpunkt steht nur zur Verfügung, wenn Text markiert ist.

Einfügen Text aus der Zwischenablage an der Cursorposition einfügen. Dieser Menüpunkt steht nur zur Verfügung, wenn Text in der Zwischenablage vorhanden ist.

Symbol einfügen Ein Symbol aus den Untermenüs an der Cursorposition einfügen.

Text formatieren Den markierten Text formatieren: Fettschrift, Kursivschrift, oder Unterstrichen.

Text kopieren nach Den gesamten Text in einen anderen Sprachbereich kopieren. Der Text wird an den evt. bereits vorhandenen Text angefügt.

Rückgängig Letzte Aktion im Textbereich zurücknehmen (außer Formatierungen).

Wiederherstellen Vorhergehende Rücknahme (⇒ Rückgängig) wieder aufheben.

Löschen Den gesamten Text löschen.

Zurückkehren Den ursprünglichen Text - entweder den Starttext, oder den Text nach dem letzten Speichern - wiederherstellen.

Sprache wechseln Andere Sprache auswählen.

Sprache hinzufügen Eine Sprache hinzufügen.

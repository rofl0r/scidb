CQL - Figurenbezeichner

Ein Figurenbezeichner ist ein Zeichenkette, die eine Menge von Figuren beschreibt, die sich auf den Schachfeldern befinden. Ein Figurenbezeichner umfasst einen Figurenartbezeichner gefolgt von einem Felderbezeichner. Der Figurenartbezeichner kennzeichnet eine Menge von Figuren, und der Felderbezeichner kennzeichnet eine Menge von Feldern. Die Menge der Figuren kann die "Leerfeldfigur" beinhalten, die ein leeres Feld beschreibt.

Ein Figurenbezeichner findet eine Position, wenn es in der Menge der Felder, die mit dem Figurenbezeichner bestimmt wird, ein Feld gibt mit einer Figur, die im Figurenartbezeichner enthalten ist.

Beispiele für den Figurenbezeichner

Der Figurenbezeichner ♖a2 kennzeichnet einen weißen Turm auf a2. Der Figurenartbezeichner ist ♖, und der Felderbezeichner ist a2. Er findet jede Position mit einem weißen Turm auf a2.

Der Figurenbezeichner ♝c1 kennzeichnet einen schwarzen Läufer auf c1. Der Figurenartbezeichner ist ♝, und der Felderbezeichner ist c1. Er findet jede Position mit einem schwarzen Läufer auf c1.

Der Figurenbezeichner [♖♝]d1 kennzeichnet wahlweise einen weißen Turm oder einen schwarzen Läufer auf d1. Der Figurenartbezeichner ist [♖♝], und der Felderbezeichner ist d1. Er findet jede Position entweder mit einen weißen Turm oder mit einen schwarzen Läufer auf d1.

Der Figurenbezeichner [♕♜][d1,h3] kennzeichnet wahlweise eine weiße Dame oder einen schwarzen Turm auf d1 oder h3. Der Figurenartbezeichner ist [♕♖], und der Felderbezeichner ist [d1,h3]. Er findet jede Position entweder mit einer weißen Dame oder einem schwarzen Turm auf d1 oder h3. (Die Position kann auch eine weiße Dame oder schwarzen Turm auf beiden Feldern d1 und h3 haben.)

Standardfigurenartbezeichner

♔ weißer König
♚ schwarzer König
♕ weiße Dame
♛ schwarze Dame
♖ weißer Turm
♜ schwarzer Turm
♗ weißer Läufer
♝ schwarzer Läufer
♘ weißer Springer
♞ schwarzer Springer
♙ weißer Bauer
♟ schwarzer Bauer

Zusätzliche Figurenartbezeichner

. Leerfeld
○ beliebige weiße Figur
● beliebige schwarze Figur
△ weiße Schwerfigur
▲ schwarze Schwerfigur
▽ weiße Leichtfigur
▼ schwarze Leichtfigur
◑ irgendeine Figur
? irgendeine Figur oder ein Leerfeld

Verwandelte Figuren

Des weiteren gibt es noch eine Kennzeichnung von Figuren, die aus einer Bauernumwandlung hervorgingen: [♕~♛~] kennzeichnet eine weiße oder schwarze Dame, die aus einer Bauernumwandlung hervorging. Es kann auch explizit spezifiziert werden, daß eine Figur nicht aus einer Bauerumwandlung hervorging, in diesem Fall wird [♕~~♛~~] verwendet. (Die Kennzeichnung von verwandelten Figuren ist insbesondere für die Schachformen Einsetzschach und Tandemschach interessant, aber auch für alle anderen Schachformen verwendbar.)

Beispiele

◑~ kennzeichnet eine beliebige, aus einer Bauerumwandlung hervorgegangene Figur
[♕~♛~][a1,h1,a8,h8] kennzeichnet eine Dame auf einem Eckfeld, die aus einer Bauernumwandlung hervorging,
◑~~ kennzeichnet eine beliebige Figur, die nicht aus einer Bauernumwandlung hervorgegangen ist
[♕~~♛~~][a1,h1,a8,h8] kennzeichnet eine Dame auf einem Eckfeld, die nicht aus einer Bauernumwandlung hervorging,

Figurenartbezeichner markieren

Überall wo ein Figurenartbezeichner auftritt kann auch ein Markierungsname, beginnend mit dem Zeichen $, auftreten. Diese Figurenart vertritt exakt die Figur, die durch den Markierungnamen vertreten wird. Für weitere Informationen siehe Seite Markierung.

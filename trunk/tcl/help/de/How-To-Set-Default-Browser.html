Festlegung des Standard-Browsers

Es existieren vier Möglichkeiten um den Standard-Browser, mit dem die externen Verweise geöffnet werden, festzulegen:

  1. Scidb benutzt den Wert der Umgebungsvariable BROWSER zur Bestimmung des Standard-Browsers. Beispiel:

    export BROWSER=/usr/bin/firefox

  2. Es wird der Verweise "x-www-browser" benutzt. Unter Debian/Ubuntu kann dieser Verweise mittels

    sudo update-alternatives --config x-www-browser

    gesetzt werden.

  3. Die Datei "~/.htmlviewrc" wird nach einem Eintrag der Form "X11BROWSER=/usr/bin/firefox" durchsucht.

  4. Es wird der zuerst auffindbare Browser aus der folgenden Liste verwendet: iceweasel, firefox, google-chrome, iexplorer, opera, konqueror, epiphany, galeon, mosaic, amaya, browsex.

Die erste der vier Möglichkeiten, die erfolgreich ist, bestimmt den Standard-Browser. Schlagen alle fehl, kann kein Browser gestartet werden und es erfolgt eine Fehlermeldung.

Konsolen-Browser werden nicht von Scidb verwendet, es muß also ein X-Browser (graphischer Browser) bestimmt werden.

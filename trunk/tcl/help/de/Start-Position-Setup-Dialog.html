Startpositionseingabe

Dieser Dialog vereinfacht die Eingabe einer Schach-960 bzw. Shuffle-Schach-Startposition. Dazu werden einfach nur die Figuren der weißen Grundreihe aufgestellt. Die OK-Schaltfläche wird nur dann freigegeben, wenn eine gültige Startposition vorliegt. Ist die Nummer der Startposition bekannt, so sollte der Stellungseingabe-Dialog zur direkten Eingabe der Positionsnummer vorgezogen werden.

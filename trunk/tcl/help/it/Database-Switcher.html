Il Selezionatore di Database

Il selezionatore permette di cambiare facilmente tra più database. Il database attivo sarà evidenziato con uno sfondo giallo. All databases not belonging to the chess variant of the active database are disabled (white background).

Premendo il tasto centrale del mouse (o la rotella) apparirà una finestra informativa.

Fuinzionalità del selezionatore di Database:

  • Amministrazione
  • Maintenance
    •
  • Trascinamento (Drag & Drop)

Amministrazione

Per l'amministrazione del database è disponibile un menu contestuale:

Proprietà Apre una finestra che mostra diverse informazioni sul database.

Esporta Apre un'interfaccia per l'esporazione del database in altri formati database o documenti.
If this database contains several chess variants (only possible in case of Clipbase or PGN files) only the games of the active chess variant will be exported.

Importa Database I database da importare saranno selezionati con la interfaccia di selezione del file. Tutte le partite saranno aggiunte alla fine del database. È possibile selezionare più database allo stesso tempo.

Importa file PGN Apre la interfaccia di selezione file per la selezione di file PGN per l'importazione. È possibile scegliere diversi file PGN simultaneamente. Le partite saranno aggiunte alla fine del database.

Maintenance This sub-menu provides functionality for the maintenance of databases (⇒ Maintenance).

Crea Archivio Crea un archivio database. La finestra di selezione file si aprirà per permettere la selezione del name dell'archivio.

Chiudi Il database selezionato sarà chiuso. Se ci sono cambiamenti nelle partite non ancora salvati, un avviso apparirà prima che il databse sia chiuso.
Chiudere il Clipbase non è possibile.

Sola Lettura Imposta il database per la sola lettura o scrittura. Questa possibilità è disponibile solo per database nei formati Scidb e Scid (.sci, .si4, .si3).
File PGN e Chessbase sono sempre in sola-lettura.
Il Clipbase non può essere impostato per la sola lettura.

Apri Database Apre un altro database.

Apri Recente Apre un database dallo storico dei database aperti.

Nuovo Database Crea un nuovo database. Solo database in formato Scidb o Scid (.sci, .si4, .si3) possono essere creati. If the database should store games of any chess variant please use the next menu item.

Nuovo Database Create a new database of any chess variant. Only Scidb databases can be created in this case.

Grandezza simbolo Seleziona la grandezza del simbolo per l'icona del database.

Aiuto per il Selezionatore di Database Apri la finestra di aiuto riguardante il selezionatore di database.

Svuota Clipbase
Questa funzione sarà disponibile solo per il Clipbase.

Maintenance

This sub-menu provides functionality for the maintenance of databases.

Cambia Icona Seleziona un'icona per il database.
Questo non è possibile per il Clipbase.

Modifica Descrizione Crea o cambia la descrizione per questo database.
Questo non è possibile per il Clipbase.

Strip Move Information Remove a selection of move information (for example "Elapsed Move Time") from database. This is useful, for example, to remove subsequently unwanted information (here "Elapsed Milliseconds") after import of FICS games.

Strip PGN Tags This feature scancs the database for extra PGN tags (such as "WhiteTeam"). Then, one may strip some of these extra tags from the database. Removing unwanted tags will accalerate position search, but not before a compaction ((⇒ Compresso) of the database is done.

Compresso Compact this database, this means that all games marked for deletion wil be deleted physically. You cannot revert this action. This menu item is disabled if the database does not contain any game with a deletion marker.

Trascinamento (Drag & Drop)

Il database può essere aperto con il trascinamento di una lista URI. Se una lista URI (lista di file) – o di un singolo file – sarà portato su un selezionatore di database, uno sfondo colorato sarà indicherà che il trascinamento di questo elemento è possibile. In questo modo potrai parire i database con l'aiuto di un gestore file esterno.

The list of files may also be dropped over a database field. In this case the games will be copied into the destination database.

To copy all games (of one chess variant) of one database to another, drag the games while holding the left mouse button and drop the games into the destination field. It is required that the destination database is writeable and not disabled. This is an important way to manage and copy games within databases.

Il Selezionatore di Database

Il selezionatore permette di cambiare facilmente tra più database. Il database attivo sarà evidenziato con uno sfondo giallo. Tutti i database che non appartengono alla variante di scacchi del database attivo sono disabilitati (sfondo bianco).

Premendo il tasto centrale del mouse (o la rotella) apparirà una finestra informativa.

Fuinzionalità del selezionatore di Database:

  • Amministrazione
  • Manutenzione
  • Trascinamento (Drag & Drop)

Amministrazione

Per l'amministrazione del database è disponibile un menu contestuale:

Proprietà Apre una finestra che mostra diverse informazioni sul database.

Esporta Apre un'interfaccia per l'esporazione del database in altri formati database o documenti.
Se il database contiene diverse varianti di scacchi (solo possibile nel caso di un Clipbase o file PGN) solo le partite relative alla variante attiva saranno esportate.

Importa Database I database da importare saranno selezionati con la interfaccia di selezione del file. Tutte le partite saranno aggiunte alla fine del database. È possibile selezionare più database allo stesso tempo.

Importa file PGN Apre la interfaccia di selezione file per la selezione di file PGN per l'importazione. È possibile scegliere diversi file PGN simultaneamente. Le partite saranno aggiunte alla fine del database.

Manutenzione Questo sotto-menu fornisce funzionalità per il mantenimento dei database (⇒ Manutenzione).

Crea Archivio Crea un archivio database. La finestra di selezione file si aprirà per permettere la selezione del name dell'archivio.

Chiudi Il database selezionato sarà chiuso. Se ci sono cambiamenti nelle partite non ancora salvati, un avviso apparirà prima che il databse sia chiuso.
Chiudere il Clipbase non è possibile.

Sola Lettura Imposta il database per la sola lettura o scrittura. Questa possibilità è disponibile solo per database nei formati Scidb e Scid (.sci, .si4, .si3).
File PGN e Chessbase sono sempre in sola-lettura.
Il Clipbase non può essere impostato per la sola lettura.

Apri Database Apre un altro database.

Apri Recente Apre un database dallo storico dei database aperti.

Nuovo Database Crea un nuovo database. Solo database in formato Scidb o Scid (.sci, .si4, .si3) possono essere creati. Se il database contiene partite di una variante eterodossa degli scacchi, per favore usa il prossimo elemento nel menu.

Nuovo Database Crea un nuovo database di qualsiasi variante eterodossa degli scacchi. Solo database con formato Scidb possono essere creati in questo caso.

Grandezza simbolo Seleziona la grandezza del simbolo per l'icona del database.

Aiuto per il Selezionatore di Database Apri la finestra di aiuto riguardante il selezionatore di database.

Svuota Clipbase
Questa funzione sarà disponibile solo per il Clipbase.

Manutenzione

Questo sotto-menu fornisce funzionalità di manutenzione del database.

Cambia Icona Seleziona un'icona per il database.
Questo non è possibile per il Clipbase.

Modifica Descrizione Crea o cambia la descrizione per questo database.
Questo non è possibile per il Clipbase.

Rimuovi Informazioni Mossa Rimuovi un'informazione mossa selezionata (per esempio "Tempo per mossa utilizzato") dal database. Questo è utile, ad esempio, per rimuovere informazioni non desiderate (qui "Tempo utilizzato in millisecondi") dopo l'importazione di partite da FICS.

Rimuovi Tag PGN Questa funzionalità scansiona il database per ulteriori tag PGN (come "WhiteTeam"). Quindi, si possono rimuovere questi ulteriori tag dal database. Rimuovendo tag non desiderati accelerarerà la ricerca per posizione, ma non prima della compressione ((⇒ Compresso) del database.

Compresso Comprime questo database, questo significa che tutte le partite segnate come da eliminare saranno eliminate fisicamente. Non si può tornare indietro. Questo elemento menu è disabilitato se non ci sono partite segnate per l'eliminazione.

Trascinamento (Drag & Drop)

Il database può essere aperto con il trascinamento di una lista URI. Se una lista URI (lista di file) – o di un singolo file – sarà portato su un selezionatore di database, uno sfondo colorato sarà indicherà che il trascinamento di questo elemento è possibile. In questo modo potrai parire i database con l'aiuto di un gestore file esterno.

La lista di file può anche essere trascinata nell'area del database. In questo caso le partite saranno copiate nel database di destinazione.

Per copiare tutte le partite (o una variante degli scacchi) da un database all'altro, trascina le partite tenendo premuto il tasto sinistro del mouse e lascia le partite nell'area di destinazione. È richiesto che il database di destinazione abbia permessi di scrittura e non disabilitato. Questo è un modo importanto per gestire e copiare partite tra database.

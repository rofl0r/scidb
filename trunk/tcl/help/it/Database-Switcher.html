Il Selezionatore di Database

Il selezionatore permette di cambiare facilmente tra più database. Il database attivo sarà evidenziato con uno sfondo giallo.

Premendo il tasto centrale del mouse (o la rotella) apparirà una finestra informativa.

Per l'amministrazione del database è disponibile un menu contestuale:

Proprietà Apre una finestra che mostra diverse informazioni sul database.

Esporta Apre un'interfaccia per l'esporazione del database in altri formati database o documenti.

Importa file PGN Apre la interfaccia di selezione file per la selezione di file PGN per l'importazione. È possibile scegliere diversi file PGN simultaneamente. Le partite saranno aggiunte alla fine del database.

Chiudi Il database selezionato sarà chiuso. Se ci sono cambiamenti nelle partite non ancora salvati, un avviso apparirà prima che il databse sia chiuso.
Chiudere il Clipbase non è possibile.

Cambia Icona Seleziona un'icona per il database.
Questo non è possibile per il Clipbase.

Modifica Descrizione Crea o cambia la descrizione per questo database.
Questo non è possibile per il Clipbase.

Sola Lettura Imposta il database per la sola lettura o scrittura. Questa possibilità è disponibile solo per database nei formati Scidb e Scid (.sci, .si4, .si3).
File PGN e Chessbase sono sempre in sola-lettura.
Il Clipbase non può essere impostato per la sola lettura.

Nuovo Database Crea un nuovo database. Solo database in formato Scidb o Scid (.sci, .si4, .si3) possono essere creati.

Apri Database Apre un altro database.

Apri Recente Apre un database dallo storico dei database aperti.

Grandezza simbolo Seleziona la grandezza del simbolo per l'icona del database.

Svuota Clipbase
Questa funzione sarà disponibile solo per il Clipbase.

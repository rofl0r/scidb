Help

Volunteers are welcome for:
  • Testing Scidb
  • Translation into other languages
  • Writing help pages
    •  Please send a message to the author, so that this tasks can be coordinated.

    How a message can be sent:

    1. A membership in Sourceforge is required (Account Registration). The membership is free of charge and duties.

    2. Open Send an Email, log in with your member name, and send the message.

    Links

    Scidb users may find the following websites interesting:

  • Tracker: Feature Requests
    Send feature requtests; please consider the existing development roadmap.

  • Tracker: Bugs
    Report problems with Scidb.

  • Reviews for Scidb
    This website offers the opportunity for a review.

  • Scidb Guestbook
    At this site the guest book is available.

  • ftp://ftp.cis.uab.edu/pub/hyatt/
    The legendary chess engine Crafty. The subdirectory there named TB has many Nalimov-format tablebase files that you can use in several chess programs and in (a later version of) Scidb.

  • The Week in Chess
    An excellent weekly bulletin of international chess games in available in PGN format.

  • Standard: Portable Game Notation Specification and Implementation Guide
    The PGN Standard, created by Steven J. Edwards in 1994. This website explains the PGN and EPD formats in detail.

  • ECO-Code
    An overview about the ECO code.

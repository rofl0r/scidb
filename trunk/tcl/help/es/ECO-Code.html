Código ECO

Código ECO (Enciclopedia de Aperturas de Ajedrez por sus siglas en inglés [Encyclopedia of Chess Openings]) es el nombre de un sistema utilizado internacionalmente para la sistematización de las aperturas de ajedrez. Las variantes se presentan y se evalúan mediante símbolos, sin utilizar palabras, de modo que puedan ser comprendidas en cualquier idioma.

  • Volumen A: Aperturas de Flanco
  • Volumen B: Aperturas semiabiertas
  • Volumen C: Aperturas abiertas
  • Volumen D: Aperturas cerradas y semicerradas
  • Volumen E: Defensas Indias

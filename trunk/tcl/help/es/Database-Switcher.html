El intercambiador de bases de datos

El intercambiador de bases de datos provee una navegación fácil entre las bases de datos. La base de datos activa se resaltará con un trasfondo amarillo. All databases not belonging to the chess variant of the active database are disabled (white background).

Al apretar el botón del medio del ratón (i.e. la rueda) se mostrará una caja de información.

Funciones del intercambiador de bases de datos:

  • Administración
  • Maintenance

  • Arrastrar & Soltar

Administración

Se encuentra disponible un menú contextual para la administración de la base de datos:

Propiedades Abre una caja que muestra información variada sobre la base de datos.

Exportar Abre el diálogo exportar base para exportar la base a otros formatos de base de datos o formatos de documento.
If this database contains several chess variants (only possible in case of Clipbase or PGN files) only the games of the active chess variant will be exported.

Importar archivos PGN Abre elegir archivo para la selección de archivos PGN para importar. Es posible seleccionar múltiples archivos PGN simultáneamente. Las partidas se agregarán al final de la base de datos.

Importar Bases de Datos The databases for import will be selected with the file selection dialog. All games will be appended at the end of the database. You may select multiple databases simultaneously.

Maintenance This sub-menu provides functionality for the maintenance of databases (⇒ Maintenance).

Crear Archivo Crea un Archivo de bases de datos. Se abrirá Elegir archivo para seleccionar un nombre de archivo.

Cerrar La base de datos seleccionada se cerrará. Si hay partidas cambiadas en la base que aún no se han guardado se mostrará un aviso de seguridad antes de cerrar la base.
No es posible cerrar la Base temporal.

Sólo lectura Establecer esta base como de sólo-lectura o editable. Esto sólo es posible para las bases Scidb y Scid (.sci, .si4, .si3).
Los archivos PGN y las bases ChessBase siempre son de sólo-lectura.
La Base temporal no puede establecerse como de sólo-lectura.

Abrir Base Abre otra base de datos.

Abrir Recientes Abre una base de datos a través del historial de bases de datos.

Nueva Base Crea una nueva base de datos. Sólo pueden crearse bases Scidb y Scid (.sci, .si4, .si3). If the database should store games of any chess variant please use the next menu item.

Nueva Base Create a new database of any chess variant. Only Scidb databases can be created in this case.

Tamaño del símbolo Selecciona el tamaño de los símbolos de base de datos.

Maintenance

This sub-menu provides functionality for the maintenance of databases.

Cambiar ícono Selecciona un ícono para esta base.
Esto no es posible para la Base temporal.

Editar Descripción Crea o cambia la descripción para esta base.
Esto no es posible para la Base temporal.

Strip Move Information Remove a selection of move information (for example "Elapsed Move Time") from database. This is useful, for example, to remove subsequently unwanted information (here "Elapsed Milliseconds") after import of FICS games.

Strip PGN Tags Not yet implemented.

Compactar Compactará la base de datos actual; esto significa que todas las partidas marcadas para su eliminación se borrarán físicamente y no podrá revertir esta acción. Este ítem no está habilitado si la base de datos no contiene ninguna partida con un marcador de eliminación.

Arrastrar & Soltar

Una base de datos puede abrirse soltando una lista URI. Si se arrastra una lista URI (lista de archivos) – o un único archivo – sobre el intercambiador de bases de datos, un trasfondo coloreado indicará que es posible soltar dicho ítem. De esta manera usted podrá abrir bases de datos con la ayuda de un administrador de archivos externo.

The list of files may also be dropped over a database field. In this case the games will be copied into the destination database.

To copy all games (of one chess variant) of one database to another, drag the games while holding the left mouse button and drop the games into the destination field. It is required that the destination database is writeable and not disabled. This is an important way to manage and copy games within databases.

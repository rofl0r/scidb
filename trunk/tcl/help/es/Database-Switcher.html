El intercambiador de bases de datos

El intercambiador de bases de datos provee una navegación fácil entre las bases de datos. La base de datos activa se resaltará con un trasfondo amarillo.

Al apretar el botón del medio del ratón (i.e. la rueda) se mostrará una caja de información.

The functionalities of the database switcher:

  • Administration
  • Drag & Drop

Administration

Se encuentra disponible un menú sensible al contexto para la administración de la base de datos:

Propiedades Abre una caja que muestra variada información sobre la base de datos.

Exportar Abre un diálogo para exportar base a otros formatos de base de datos o formatos de documento.

Importar archivos PGN Abre el diálogo de selección de archivo para la selección de archivos PGN para importar. Es posible seleccionar múltiples archivos PGN simultáneamente. Las partidas se agregarán al final de la base de datos.

Crear Archivo Create a database archive. The file selection dialog will be opened for the selection of an archive name.

Cerrar La base de datos seleccionada se cerrará. Si hay partidas cambiadas en la base que aún no se han guardado se mostrará un aviso de seguridad antes de cerrar la base.
No es posible cerrar Clipbase.

Compactar Compact this database, this means that all games marked for deletion wil be deleted physically. You cannot revert this action. This menu item is disabled if the database does not contain any game with a deletion marker.

Cambiar ícono Selecciona un icono para esta base.
Esto no es posible para Clipbase.

Editar Descripción Crea o cambia la descripción para esta base.
Esto no es posible para Clipbase.

Sólo lectura Establecer esta base como de sólo-lectura o editable. Esto sólo es posible para las bases Scidb y Scid (.sci, .si4, .si3).
Los archivos PGN y las bases ChessBase siempre son de sólo-lectura.
Clipbase no puede establecerse como de sólo-lectura.

Nueva Base Crea una nueva base de datos. Sólo pueden crearse bases Scidb y Scid (.sci, .si4, .si3).

Abrir Base Abre otra base de datos.

Abrir Recientes Abre una base de datos a través del historial de bases de datos.

Tamaño del símbolo Selección del tamaño de los símbolos de base de datos.

Vaciar Base temporal
Esta entrada de menú sólo se proporcionará para Clipbase.

Drag & Drop

Database can be opened with the drop of an URI list. If an URI list (list of files) – or a single file – will be dragged over the database switcher, a colored background will indicate that a drop of this item is possible. In this way you may open databases with the help of an external file manager.

El Explorador de partidas

El Explorador de partidas permite seleccionar cualquier partida de una lista de partidas abierta. El Explorador está ligado a dicho listado de partidas. No se podrá cerrar el listado de partidas, ni cargar otra partida, mientras el Explorador continúe abierto.

El Explorador de partidas muestra dichas partidas en un formato sencillo, es decir, sin anotaciones, comentarios o subvariantes. Está dirigido especialmente a la visualización de un Listado de partidas.

El encabezado de la partida contiene tres campos: el Torneo y los Nombres del jugador. Manteniendo apretado el botón central del ratón aparece un cuadro de información. Al cliquear sobre el Nombre de un jugador aparece la Ficha de dicho jugador.

If the color of the game header is changing from blue to red, this will signal an intermediate change of the game in the database. The game can be re-loaded to see the actual game (⇒ Reload game inside context-sensitive menu).

Si el puntero del ratón se encuentra fuera de la Ventana de texto (de la partida), pero aún dentro del cuadro de diálogo, girando la rueda se pueden oir las sucesivas jugadas.

While keeping the middle mouse button pressed on a game move the associated board position will be displayed. Holding Mayúsculas during the click will show only the pawn structure.

The browser is providing the following context-sensitive menu:

Cargar partida into editor Load the actual game into the game editor. The actual game in game editor will be replaced with this game, so far the editor isn't locked or the game is modified. Otherwise a new pane for this game will be opened in editor.

Reload game Update the actual game by re-loading. This menu entry is active only if the current game is differing from the game in database due to an intermediate change. A red coloured game header (normally the game header is blue) will signal that the current game is differing from the game in database.

Ir a la siguiente partida Load next game in underlying game list.

Ir a la partida anterior Load previous game in underlying game list.

Ir a la última partida Load last game in underlying game list.

Ir a la primera partida Load first game in underlying game list.

Agrandar el tablero Increase the board size, and the dialog size as a consequence.

Agrandar el tablero Decrease the board size, and the dialog size as a consequence.

Maximizar el tamaño del tablero Maximize the board size, and the dialog size as a consequence (but no full-screen).

Minimizar el tamaño del tablero Minimize the board size, and the dialog size as a consequence.

Pantalla completa Use full-screen, this means use the whole screen. This function is not available for all screen resolutions.

Aumentar el tamaño de la fuente Increase the font for displaying text.

Disminuir el tamaño de la fuente Decrease the font for displaying text.

Estilo columna Switch column format in move window on/off.

Anotación de Jugadas Choose the move notation in associated sub-menu:

Algebraica Corta   ♘f3
Algebraica Larga   ♘g1−f3
Algebraica   g1f3
Inglés   N−KB3
Correspondencia   7163
Telegrafico   KAHI

Personalizar Salida de texto Opens a dialog which allows an extensive configuration of game display.

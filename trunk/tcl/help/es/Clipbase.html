La base de datos Clipbase

Scidb proporciona una base de datos temporal (Clipbase) abierta al inicio del programa, la cual se comporta como cualquier otra base de datos excepto por el hecho de que sólo existe en la memoria y no tiene archivos para su almacenaje permanente. Esto significa que la capacidad de esta base de datos es limitada. El nombre predefinido para esta base de datos es Base temporal.

La Base temporal no puede cerrarse. Existe una función de menú para vaciarla (⇒ Vaciar Base temporal).

The Clipbase keeps data in Scidb's native format and is an exception, because it may contain various variants concurrently. But administrative operations (⇒ Maintenance) can be applied to only one variant.

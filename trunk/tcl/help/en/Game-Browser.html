Game Browser

The game browser will be opened with a game from a game list. The browser is bound to this game list. If the game list is closed while the browser is open, no more games can be loaded inside this browser.

The game brower displays games in a simple form, this means without annotations and sub-variants. The browser is especially suitable for displaying list of games.

The game header contains three buttons: the event and both player names. While keeping the middle mouse button pressed on any button an information box will be displayed. Clicking a player name is opening a player card.

If the color of the game header is changing from blue to red, this will signal an intermediate change of the game in the database. The game can be re-loaded to see the actual game (⇒ Reload game inside context-sensitive menu).

While the mouse is not hovering the move window, but still hovering the dialog, the mouse wheel can be used to play moves forward and backward.

While keeping the middle mouse button pressed on a game move the associated board position will be displayed. Holding Shift during the click will show only the pawn structure.

The browser is providing the following context-sensitive menu:

Load game into editor Load the actual game into the game editor. The actual game in game editor will be replaced with this game, so far the editor isn't locked or the game is modified. Otherwise a new pane for this game will be opened in editor.

Reload game Update the actual game by re-loading. This menu entry is active only if the current game is differing from the game in database due to an intermediate change. A red coloured game header (normally the game header is blue) will signal that the current game is differing from the game in database.

Goto next game Load next game in underlying game list.

Goto previous game Load previous game in underlying game list.

Goto last game Load last game in underlying game list.

Goto first game Load first game in underlying game list.

Increase board size Increase the board size, and the dialog size as a consequence.

Increase board size Decrease the board size, and the dialog size as a consequence.

Maximize board size Maximize the board size, and the dialog size as a consequence (but no full-screen).

Minimize board size Minimize the board size, and the dialog size as a consequence.

Full-Screen Use full-screen, this means use the whole screen. This function is not available for all screen resolutions.

Increase Font Size Increase the font for displaying text.

Decrease Font Size Decrease the font for displaying text.

Use Column Style Switch column format in move window on/off.

Move Notation Choose the move notation in associated sub-menu:

Short Algebraic   ♘f3
Long Algebraic   ♘g1−f3
Algebraic   g1f3
English   N−KB3
Correspondence   7163
Telegraphic   KAHI

Customize Text Output Opens a dialog which allows an extensive configuration of game display.

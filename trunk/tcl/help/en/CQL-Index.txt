TITLE CQL - Chess Query Language

<p>
Chess Query Language (CQL) was designed to allow researchers, authors, and
players to search for games, problems, and studies that match specific themes
(<see/> <a href="http://www.rbnn.com/cql/">CQLManual</a>).
</p>

<p>
Scidb is using an extended set of CQL as a basis for all complex search
functions.
</p>

<p>
Note that some parts of this documentation are taken from
<a href="http://www.rbnn.com/cql/">CQLManual</a>.
</p>

<p>
We are using graphical figurine for the notation of chess pieces, due to
internalization reasons. The CQL editor is working with graphical figurine,
although the CQL files will be saved with the corresponding English letters:
</p>

What's new?

  • Game history

  • Help dialog (with a few help pages)

  • New file selection box with extraneous functions:

    • Delete database
    • Rename database
    • Duplicate database
    • Create new folder
    • Selection of database encoding

  • Automatic detection of database encoding

  • Enhanced Scidb database format; now it exists a new comment type: move information. This provides a compact storage and, in a future version, a facilitated input.

  • Automatic upgrade to the new database format

  • Revised export dialog

  • Undo/redo in comment editor

  • Improved behaviour of the HTML window

    • Double buffering for flicker-free display
    • Automatic hyphenation

  • Accelerated popup of the game save dialog

  • Improved tournament table:

    • Additional scoring system "Bilbao" (three points for a win)
    • Additional tie-breaks

  • The export to .si3/.si4 now offers a selection of the PGN-Tags (a restriction of the tags is accelerating the position tree)

  • Selection dialog of database symbols is revised

  • New feature in game editor: copy/move comments between different languages

  • Drop-down shadows (experimental)

  • Partial translation into Hungarian; the translation is still in progress

  • Ths Spanish translation is taken out temporary due to incompleteness. Volunteers for the translation: please look at Contact information

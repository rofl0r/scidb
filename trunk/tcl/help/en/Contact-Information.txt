<h2 id="Homepage">Homepage</h2>

The Scidb web page is located at
<a href="http://scidb.sourceforge.net/">http://scidb.sourceforge.net</a>.

<h2>Help</h2>

Volunteers are welcome for:

<dir>
  <li>Testing Scidb</li>
  <li>Translation into other languages</li>
  <li>Writing help pages</li>
</dir>

Please send a message to the author, so that this tasks
can be coordinated.

<box>
<p>
How a message can be sent:
</p>
<ol>
   <li><p>
      A membership in Sourceforge is required
      (<a href="https://sourceforge.net/account/registration">Account Registration</a>).
      The membership is free of charge and duties.
    </p></li>
    <li><p>
      Open <a href="https://sourceforge.net/sendmessage.php?touser=2321037">Send an Email</a>,
      log in with your member name, and send the message.
    </p></li>
</ol>
</box>
  
<h2>Links</h2>
  
<p>
Scidb users may find the following websites interesting:
</p>

<dir>
  <li><p>
    <a href="http://sourceforge.net/tracker/?atid=1294800&amp;group_id=307371&amp;func=browse"
       style="line-height: 150%;">
    Tracker: Feature Requests</a><br/>
    Send feature requtests; please consider the existing
    <a href="Development-Roadmap.html">development roadmap</a>.
  </p></li>

  <li><p>
    <a href="http://sourceforge.net/tracker/?atid=1294797&amp;group_id=307371&amp;func=browse"
       style="line-height: 150%;">
    Tracker: Bugs</a><br/>
    Report problems with Scidb.
  </p></li>

  <li><p>
    <a href="http://sourceforge.net/projects/scidb/reviews/"
       style="line-height: 150%;">
    Reviews for Scidb</a><br/>
    This website offers the opportunity for a review about Scidb.
  </p></li>

  <li><p>
    <a href="https://lists.sourceforge.net/lists/listinfo/scidb-users"
       style="line-height: 150%;">
    Subscribe Scidb Users Mailing List</a><br/>
    At this place you may do an entry into the member list of the
    "Scidb Users Mailing List". This user list provides information, news,
    and proposals. Members of this list may send messages which will be
    distributed to all members automatically.
  </p></li>

  <li><p>
    <a href="http://sourceforge.net/apps/aguestbook/scidb/sign.php"
       style="line-height: 150%;">
    Scidb Guestbook</a><br/>
    At this site the guest book is available.
  </p></li>

  <li><p>
    <a href="ftp://ftp.cis.uab.edu/pub/hyatt/"
       style="line-height: 150%;">
    ftp://ftp.cis.uab.edu/pub/hyatt/</a><br/>
    The legendary chess engine Crafty. The subdirectory there named TB has many
    Nalimov-format tablebase files that you can use in several chess programs and
    in (a later version of) Scidb. 
  </p></li>

  <li><p>
    <a href="http://tablebase.sesse.net"
       style="line-height: 150%;">
    tablebase.sesse.net</a><br/>
    This site does also provide tablebase files, including 6-men bases.
  </p></li>

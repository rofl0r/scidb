    <li><p>Automatic upgrade to the new database format</p></li>
    <li><p>Revised export dialog</p></li>
    <li><p>Undo/redo in comment editor</p></li>
    <li><p>Improved behavior of the HTML window:</p>
      <ul type="circle">
        <li>Double buffering for flicker-free display</li>
        <li>Automatic hyphenation</li>
      </ul>
    </li>
    <li><p>Accelerated popup of the game save dialog</p></li>
    <li><p>Improved tournament table:</p>
      <ul type="circle">
        <li>Additional scoring system "Bilbao" (three points for a win)</li>
        <li>Additional tie-breaks</li>
      </ul>
    </li>
    <li><p>The export to .si3/.si4 now offers a selection of the <acronym>PGN</acronym>-Tags (a restriction of the tags is accelerating the position tree)</p></li>
    <li><p>Selection dialog of database symbols is revised</p></li>
    <li><p>New feature in game editor: copy/move comments between different languages</p></li>
    <li><p>Drop-down shadows (experimental)</p></li>

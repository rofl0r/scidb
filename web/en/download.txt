<div class="block">

  <div class="header">
    Download <hr />
  </div>

	<p>
  The third public preview version is ready for download:
  <a href="http://sourceforge.net/projects/scidb/files/scidb-beta--2013-04-01--rev701.tar.gz/download">scidb-beta--2013-04-01--rev701.tar.gz</a>.
  This version is for Linux/Unix systems only. Thw windows version will be released later,
  presumably at end of year 2013 (unfortunately postponed again, the implementation takes
  a longer time than expected).
  </p>

	<p>
  Possibly you have to install the Tcl/Tk packages before Scidb can be compiled.
  On Ubuntu/Debian systems use:
  </p>

	<p>
	&nbsp;&nbsp;&gt; sudo apt-get install tcl8.5<br />
	&nbsp;&nbsp;&gt; sudo apt-get install tk8.5<br />
	</p>

	<p>Install Scidb as usual:</p>
	
	<p>
	&nbsp;&nbsp;&gt; tar -xzf scidb-beta--2013-04-01--rev701.tar.gz<br />
	&nbsp;&nbsp;&gt; cd scidb-beta--2013-04-01--rev701<br />
	&nbsp;&nbsp;&gt; ./configure<br />
	&nbsp;&nbsp;&gt; make<br />
	&nbsp;&nbsp;&gt; sudo make install
	</p>

	<p>
  After the installation succeeded <code>scidb-beta</code> is executable.
  </p>

  <p>
  Please consider the specific problems under Ubuntu editions as of version
  11.10 (see <a href="releasehistory.html">Release history</a>).
  </p>

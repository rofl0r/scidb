CONTENT

<div class="block">

  <div class="header">
    Welcome to Scidb <hr />
  </div>

  <p>
  <!-- google_ad_section_start -->
  Scidb ("Scidb is a Chess Information Data Base")
  <!-- google_ad_section_end -->
  is a chess database application for Linux/Unix and Windows. Scidb is inspired by Scid
  ("Shane's Chess Information Database"), but it is a completely new development.</p>

  <p>
  The principal features of this chess database application:
  </p>

  <dir>
    <li>
      Support of various chess variants:
      <ul>
        <li><a target="_blank" href="http://en.wikipedia.org/wiki/Three-check_chess">Three-Check chess<link/></a></li>
        <li><a target="_blank" href="http://en.wikipedia.org/wiki/Antichess">Antichess<link/></a>
            (<a target="_blank" href="http://suicidechess.ca/index.php?lg=en">Suicide<link/></a>,
            <a target="_blank" href="http://www6.chessclub.com/help/Giveaway">Giveaway<link/></a>,
            <a target="_blank" href="http://loserschess.org/">Losers<link/></a>)</li>
        <li><a target="_blank" href="http://en.wikipedia.org/wiki/Crazyhouse">Crazyhouse<link/></a></li>
        <li><a target="_blank" href="http://en.wikipedia.org/wiki/Bughouse_chess">Bughouse<link/></a>
            (in preparation)</li>
        </ul>
    </li>

    <li>
    Support of <a target="_blank" href="http://en.wikipedia.org/wiki/Chess960">Chess 960<link/></a>
    and <a target="_blank" href="http://en.wikipedia.org/wiki/Chess960">Shuffle Chess<link/></a>
    </li>

    <li>
      Support of various database and file formats:
      <dir>
        <table>
          <tr><td>Scidb:</td>     <td>.sci, .scv</td></tr>
          <tr><td>Scid:</td>      <td>.si3, .si4</td></tr>
          <tr><td>ChessBase:</td> <td>.cbh, .cbf (read-only)</td></tr>
          <tr><td>PGN:</td>       <td>.pgn, .pgn.gz, .zip</td></tr>
        </table>
      </dir>
    </li>

    <li>
      28 different SVG/TrueType chess piece sets
    </li>

    <li>
      User-configurable chess board themes (based on SVG)
    </li>

    <li>
      <a href="help/en/CQL-Index.html">CQL</a> based search functions (in development)
    </li>

    <li>
      PDF export (in preparation)
    </li>
  </dir>

</div><!-- block -->

<div class="block">

  <div class="header">
    News <hr />
  </div>

  <p>
  The third public preview version is released
  (see <a href="download.html">Download</a>).
  This version contains many, many bug-fixes, and also many novelties.
  Some time will pass until the first stable version, but it's not far anymore.
  The most important missing goals are:
  </p>

  <dir>
    <li>Flexible window layout (unfortunately very complicated)</li>
    <li>Searching and filtering functionality: search of games, position search, etc.
        The search functionality will be based on CQL (Chess Query Language;
        see <a href="help/en/CQL-Index.html">CQL Manual</a>).</li>
    <li>Multiple selections in game list</li>
    <li>Support of Bughouse games</li>
    <li>Support of variants (Crazyhouse, Antichess, Three-Check) in opening tree</li>
    <li>Input of games with keyboard</li>
  </dir>

  <p>
  With the implementation of this items the first full version will be
  released - possibly with the next version.
  </p>

  <p>
  Of course the Windows version is pending. This requires:
  </p>

  <dir>
    <li>
      A (quite complex) library to emulate the missing functions <code>fopencookie()</code>,
      and <code>open_memstream()</code> under Windows.
    </li>
    <li>
      Porting all system specific functionalities, so far completely implemented
      only under Unix - currently the porting is done partially under Windows.
    </li>
    <li>
      Intensive testing because the GUI has a different behavior under Windows.
    </li>
  </dir>

</div><!-- block -->

<div class="block">

  <div class="header">
    What's new in third version? <hr />
  </div>

  <p>
  The most important novelties are:
  </p>

  <dir>
    <li>
      Support of different chess variants: Crazyhouse, Antichess (Suicide, Giveaway, Losers),
      and Three-Check. In current version the support is still a bit experimentally, but
      with full functionality.
    </li>
    <li>
      Analysis window with support of external chess engines. Due to the lack of
      the still missing window layout a standalone window will be opened for analysis.
    </li>
    <li>
      Dialog for setup of external chess engines. Two chess engines are already
      pre-installed:
      <ul>
        <li>Stockfish (supports <i>Chess 960</i>, and <i>Three-Check</i>)</li>
        <li>Sjeng (supports Crazyhouse, Antichess, and later Bughouse)</li>
      </ul>
    </li>
    <li>
      Automatic update of supplements via the Internet, currently these are the
      player photo files.
    </li>
    <li>
      Integration of format independent database archives. Among other things
      this facilitates an easier shipping of databases via Internet.
    </li>
    <li>
      Drag&amp;Drop between databases.
    </li>
    <li>
      Mouse wheel support for all list windows and the chess board.
    </li>
    <li>
      Serious performance problem when opening ChessBase databases is solved.
    </li>
    <li>
      Merge of games, optionally takes into account move transitions.
    </li>
    <li>
      Game editor now features various settings.
    </li>
    <li>
      There exists a player dictionary (with sorting and filtering functionality)
      and an engine dictionary.
    </li>
  </dir>

  <p>
  More novelties:
  </p>

  <dir>
    <li>Move input: pressing middle mouse button (or spacebar) alternates the move suggestion</li>
    <li>New database tab: list of sites</li>
    <li>Full-screen mode tweaked</li>
    <li>Command line tool cdb2sci added for conversion of databases to Scidb format (.sci)</li>
    <li>HTML widget reworked: saving resources in a highly significant way</li>
    <li>Swedish localization integrated (thanks to Lars Ekman)</li>
    <li>Spanish translation reactivated (thanks to Juan Carlos Vásquez)</li>
    <li>Spanish help pages added (thanks to Carlos Fernando González)</li>
    <li>HTML file export of tournament table</li>
    <li>New piece sets: "Free Staunton", and "Virtual"</li>
    <li>New board themes: "Blue Theme", "Staunton", and "Virtual"</li>
    <li>Automatic detection of character set when encoding games (only for si3/si4/cbh)</li>
    <li>Game browser: full-screen mode integrated, and also maximize/minimize board</li>
    <li>File selection dialog improved; Drag&amp;Drop among other things</li>
    <li>Game-Save/Replace dialog improved</li>
    <li>Player notecards implemented</li>
    <li>Database window: docking/undocking of list tabs integrated</li>
    <li>Move input with mouse overworked (if move suggestion is disabled)</li>
    <li>Menu bar removed: instead a simple main menu exists; this is state-of-the-art,
        and the layout will be improved</li>
    <li>Compacting database (.sci only) integrated; required for the physical removal of
        games marked for deletion</li>
    <li>Tournament table: optional display of win/draw/loss</li>
    <li>Algebraic (normal, short, and long), English, Correspondence, and Telegraphic move notations available</li>
    <li>Now it is possible to import databases without to open beforehand</li>
    <li>Severe problem with Tcl version 8.6 solved</li>
    <li>More help pages</li>
    <li>Some icons newly designed</li>
  </dir>
</div><!-- block -->

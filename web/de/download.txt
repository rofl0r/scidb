TITLE

	Scidb - Download

CONTENT

<div class="block">

  <div class="header" lang="en">
    Download <hr /> 
  </div>

  <p>
  Die dritte öffentliche Vorschauversion ist bereit für den <span lang="en">Download</span>:
  <a href="http://sourceforge.net/projects/scidb/files/scidb-beta--2013-04-01--rev692.tar.gz/download">scidb-beta--2013-04-01--rev692.tar.gz</a>.
  Diese Version ist nur für Linux/Unix Betriebssysteme. Die Windows-Version wird später
  veröffentlicht, eventuell am Ende des Jahres 2013 (hat sich leider wieder verschoben, die
  Implementierung dauert doch länger als erwartet).
  </p>

  <p>
  Möglicherweise sollten zuerst die Tcl/Tk Pakete installiert werden, bevor Scidb gebaut
  werden kann. Unter Ubuntu/Debian-Distributionen wird Tcl/Tk folgendermassen installiert:
  </p>

	<p><code>
	&nbsp;&nbsp;&gt; sudo apt-get install tcl8.5<br />
	&nbsp;&nbsp;&gt; sudo apt-get install tk8.5<br />
	</code></p>

  <p>Scidb wird wie gewöhnlich gebaut:</p>
	
	<p><code>
	&nbsp;&nbsp;&gt; tar -xzf scidb-beta--2013-04-01--rev692.tar.gz<br />
	&nbsp;&nbsp;&gt; cd scidb<br />
	&nbsp;&nbsp;&gt; ./configure<br />
	&nbsp;&nbsp;&gt; make<br />
	&nbsp;&nbsp;&gt; sudo make install
	</code></p>

	<p>
  Nach einer erfolgreichen Installation steht <code>scidb-beta</code> zur Ausführung bereit.
  </p>

  <p>
  Es sollten die spezifischen Probleme unter den Ubuntu Editionen ab
  Version 11.10 beachtet werden (siehe
  <a href="releasehistory.html">Freigabehistorie</a>),
  </p>

</div>

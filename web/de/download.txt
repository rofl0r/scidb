TITLE

	Scidb - Download

CONTENT

<div class="block">

  <div class="header" lang="en">
    Download <hr /> 
  </div>

  <p>
  Die zweite öffentliche Vorschauversion ist bereit für den <span lang="en">Download</span>:
  <a href="http://sourceforge.net/projects/scidb/files/scidb-beta--2012-02-19--rev250.tar.gz/download">scidb-beta--2012-02-19--rev250.tar.gz</a>.
  Diese Version ist nur für Linux/Unix Betriebssysteme. Die Windows-Version wird später
  veröffentlicht, eventuell am Ende des Jahres 2012.
  </p>

  <p>
  Möglicherweise sollten zuerst die Tcl/Tk Pakete installiert werden, bevor Scidb gebaut
  werden kann. Unter Ubuntu/Debian-Distributionen wird Tcl/Tk folgendermassen installiert:
  </p>

	<p>
	&nbsp;&nbsp;&gt; sudo apt-get install tcl8.5<br />
	&nbsp;&nbsp;&gt; sudo apt-get install tcl8.5-dev<br />
	&nbsp;&nbsp;&gt; sudo apt-get install tk8.5<br />
	&nbsp;&nbsp;&gt; sudo apt-get install tk8.5-dev
	</p>

  <p>Scidb wird wie gewöhnlich gebaut:</p>
	
	<p>
	&nbsp;&nbsp;&gt; tar -xzf scidb-beta--2012-02-19--rev250.tar.gz<br />
	&nbsp;&nbsp;&gt; cd scidb<br />
	&nbsp;&nbsp;&gt; ./configure<br />
	&nbsp;&nbsp;&gt; make<br />
	&nbsp;&nbsp;&gt; sudo make install
	</p>

	<p>
  Nach einer erfolgreichen Installation steht 'scidb-beta' zur Ausführung bereit.
  </p>

  <p>
  Es sollten die spezifischen Probleme auf den defekten Plattformen
  <acronym>Debian</acronym>&nbsp;7.0 (<acronym>Wheezy</acronym>) und
  <acronym>Ubuntu</acronym>&nbsp;11.10 (<acronym>Oneiric</acronym>)
  beachtet werden (siehe <a href="releasehistory.html">Freigabehistorie</a>).
  </p>
